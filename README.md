# Real Calçados

Site web para Real Calçados, desenvolvido com Next.js para proposta comercial.

## Descrição

Este projeto é um website básico para a empresa Real Calçados, utilizando Next.js, TypeScript e Tailwind CSS.

## Instalação e Execução

1. Instale as dependências:

```bash
npm install
```

2. Execute o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador para ver o resultado.

## Build e Deploy

Para build:

```bash
npm run build
```

Para deploy no Vercel, consulte a [documentação de deploy do Next.js](https://nextjs.org/docs/app/building-your-application/deploying).

## Tecnologias

- Next.js
- TypeScript
- Tailwind CSS
- ESLint
